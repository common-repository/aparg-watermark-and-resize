== Description ==

= On-Fly =
Aparg Watermark and Resize designed for resizing images and placing watermark on them on fly. 
Images will be automatically resized and watermark will be placed on them when you upload images to WordPress media library. 
You can choose whether to do both actions or only one of them.

= Bulk action =
You can also perform bulk action. In this case actions will be applied to all existing images in media library. 
We recommend to make backup of your website before bulk action.

In settings page you can choose watermark image, position, size(%) and margin from borders(%). 
Also you can set max box size of images and they will be resized to given box with keeping ratio. 
Plugin is translation ready.

== Installation ==

Automatic Installation:

1. Go to 'Admin' -> 'Plugins' -> 'Add New' and search for "Aparg Watermark and Resize"

2. Click on the 'Install Now' button

3. Click on the 'Activate Plugin' link


Manual Installation:

1. Download plugin archive

2. Go to 'Admin' -> 'Plugins' -> 'Add New' and click on the 'Upload Plugin' button

3. Choose plugin archive and click on the 'Install Now' button

4. Click on the 'Activate Plugin' link
